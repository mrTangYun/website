browersync

Browsersync能让浏览器实时、快速响应您的文件更改（html、js、css、sass、less等）并自动刷新页面。更重要的是 Browsersync可以同时在PC、平板、手机等设备下进项调试。您可以想象一下：“假设您的桌子上有pc、ipad、iphone、android等设备，同时打开了您需要调试的页面，当您使用browsersync后，您的任何一次代码保存，以上的设备都会同时显示您的改动”。无论您是前端还是后端工程师，使用它将提高您30%的工作效率。
有了它，您不用在多个浏览器、多个设备间来回切换，频繁的刷新页面。更神奇的是您在一个浏览器中滚动页面、点击等行为也会同步到其他浏览器和设备中，这一切还可以通过可视化界面来控制。
官网：https://browsersync.io/
中文网站：http://www.browsersync.cn/

  • 安装命令

    npm install -g browser-sync

  • 启动命令

    • 静态网站

      部分文件：browser-sync start --server --files "css/*.css"

      所有文件：browser-sync start --server --files "**"

    • 动态网站

      如果您已经有其他本地服务器环境PHP或类似的，您需要使用代理模式。 BrowserSync将通过代理URL(localhost:3000)来查看您的网站。// 主机名可以是ip或域名

      browser-sync start --proxy "hyundai.koo7.com" "css/*.css"
